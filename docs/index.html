Clinvio Nucleus

Enterprise Java framework with 34 modules for secure, server-rendered web applications

v2.0.0 Java 21 Spring Boot 3.3+ 34 Modules

Introduction

Clinvio Nucleus is a 34-module enterprise Java framework providing UI components, persistence, security, observability, and domain-specific features for medical, logistics, cloud, and project management applications.
